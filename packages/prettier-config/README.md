# @braviadev/prettier-config

Personal Prettier configurations for Olanrewaju Toyyib projects.

## Installation

```bash
npm i -D @braviadev/prettier-config prettier-plugin-packagejson
```

Create a `prettier.config.js` file with the following content:

```js
import { defineConfig } from '@braviadev/prettier-config'

export default defineConfig({
  // Custom Prettier configuration options
})
```
